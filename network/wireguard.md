# Wireguard
https://www.wireguard.com/
Wireguard is a [[VPN]] solution for secure connection over internet or LAN
State of the art crypto
## Working
	| Alice                            | Bob
	| IP packet to 10.0.0.1            |
	| Who is 10.0.0.1?                 |
	| It is Bob                        |
	| Encrypt packet with Bob's key    |
	| Encrypted IP packet to Bob ----> |
